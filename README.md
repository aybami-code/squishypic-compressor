*Product name* : squishypic-compressor


Squishypic is a lightweight, browser-based image compressor that reduces JPG, PNG, and WebP files instantly — with zero uploads. Everything runs locally for maximum speed and privacy.

Perfect for designers, students, content creators, and anyone who needs quick image optimization.




 *Features*

 Instant compression (JPG, PNG, WebP)

 Adjustable output quality

 Live preview (before/after)

 Shows original vs compressed file size

 One-click download

 Fully responsive UI

 100% offline and privacy-friendly

 No backend required





*Tech Stack*

HTML

CSS

JavaScript (Canvas API)





*How to Use*

1. Open the site.


2. Upload an image.


3. Adjust the compression level.


4. Preview the result.


5. Download the optimized file.
